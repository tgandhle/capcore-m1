This is the M1 capability core rendered in the browser: real authorization algebra (per-grant scope check, union of alternative grants, most-restrictive approval, fail-closed), real deny > require-approval > allow precedence including an explicit platform-deny policy. No model or tool network calls; fonts are system-local. Identity (tenant · principal · run) is supplied by a trusted context, never read from the model's proposal. Containment shown here is evidence from a decision trace, not proof: these properties are intended to be backed by a property-based test suite (the Python M1 core), which is where the enumerable claims get exhaustively checked.
